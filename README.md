## ■ 作成したアプリ

### 1. カフェ空席状況アプリ（Firebase完全同期版） ★PICK UP
店側（管理）とお客様側（閲覧・予約）をクラウドで繋ぐリアルタイムWebアプリ。

* **フルクラウド同期**: すべての席情報（空席・予約・利用中）をFirestoreで一元管理。店側が「利用中」に切り替えた瞬間、客側の画面も**リロードなしで即座に**変化する「双方向リアルタイム更新」を実現。
* **高度なデータ連携**: `updateDoc` や `setDoc` を使い分け、特定の座席ステータスのみをピンポイントで更新する効率的なデータ操作。
* **JavaScript Module**: スコープを意識した最新のモジュール形式（type="module"）を採用。
* **完全レスポンシブ**: ドイツでの生活環境（スマホ利用）を想定し、モバイルファーストで設計。

### 2. 社員管理アプリ（Web版）
ブラウザ上で社員情報の操作・検索ができる管理ツール（localStorage対応）。
