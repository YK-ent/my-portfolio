## ■ 作成したアプリ

### 1. カフェ空席状況アプリ（Firebase完全同期版） ★PICK UP
店側とお客様側をクラウドで繋ぐリアルタイムWebアプリ。
- **URL**: https://cafe-app-firebase-5fb26.web.app

* **世界公開済み**: Firebase Hostingによりデプロイ。PC・スマホ間でのマルチデバイス同期を確認済み。
* **堅牢なデータ管理**: `getDoc` による重複チェックに加え、ステータスに応じたボタンの制御（`disabled`）により、データの矛盾を徹底排除。
* **実務を意識したUX**: `confirm` ダイアログによる誤操作ガード、Enterキー入力対応など、現場のスピード感を意識した設計。
* **フルクラウド同期**: Firestoreによる双方向リアルタイム反映。
