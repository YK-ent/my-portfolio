# My Portfolio

JavaScriptとGitの学習記録をまとめたポートフォリオです。  
基礎から小さなアプリ作成まで、学習過程を記録しています。

---

# 学習内容

このリポジトリでは以下の内容を学習しています。

* Git / GitHub の基本操作
* JavaScript 基礎（変数・配列・オブジェクト）
* 配列操作（forEach / map / filter / sort）
* 関数と条件分岐
* 簡単なアプリ作成（社員管理アプリなど）
* アプリ拡張（追加・削除・検索・メニュー機能）

---

# 使用技術

* Git
* GitHub
* JavaScript (ES6)
* Node.js（JavaScript実行環境）
