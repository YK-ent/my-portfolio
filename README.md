## ■ 作成したアプリ

### 1. カフェ空席状況アプリ（Firebase完全同期版） ★PICK UP
店側（管理）とお客様側（閲覧・予約）をクラウドで繋ぐリアルタイムWebアプリ。

* **フルクラウド同期**: すべての席情報をFirestoreで一元管理。店側が操作した瞬間に客側の画面も**リロードなしで即座に**変化する双方向同期。
* **データバリデーション**: `getDoc` を活用し、同一の席名が登録されるのを防ぐロジックを実装。データの重複や上書きエラーを未然に防止。
* **UX（ユーザー体験）の追求**: 店員がキーボードから手を離さずに操作できる「Enterキーでの座席追加」や、予約客の来店時に1クリックでステータスを変更できる「入店ショートカットボタン」を実装。
* **完全レスポンシブ**: ドイツでの生活環境（スマホ利用）を想定し、モバイルファーストで設計。

### 2. 社員管理アプリ（Web版）
ブラウザ上で社員情報の操作・検索ができる管理ツール（localStorage対応）。
