## ■ 作成したアプリ

### 1. カフェ空席状況アプリ（Web版） ★PICK UP
店側（管理）とお客様側（閲覧・予約）の2つの画面を持つ、リアルタイム性を意識したWebアプリ。

* **3状態管理システム**: 「空席(🟢)」「利用中(🔴)」に「予約済(🟡)」を加えた高度な状態管理。
* **双方向リアルタイム同期**: `setInterval` を活用し、店側の操作を数秒以内にお客様画面へ反映。
* **インタラクティブな予約**: 客側からの予約・キャンセル、店側からの予約解除ロジックを実装。
* **完全レスポンシブ**: モバイルファーストの視点を取り入れ、スマホでの操作性を最適化。

### 2. 社員管理アプリ（Web版）
ブラウザ上で社員情報の追加・編集・削除・検索ができる管理ツール。
* **機能**: ID/名前順ソート、重複チェック、localStorageによるデータ保存、編集モードUI。

### 3. 社員管理アプリ（CLI版）
Node.js環境で動作する、コマンドラインベースの管理ツール。
